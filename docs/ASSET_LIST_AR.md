# الملفات الثنائية المطلوبة (أضفها يدويًا)

بناءً على طلبك، لم أضف أي ملفات صوت/صور ثنائية داخل المشروع.
إذا أردت تفعيل المؤثرات الصوتية، أضف الملفات التالية:

1. `audio/start.wav`  ← صوت بدء اللعبة (عند الضغط على زر ابدأ).
2. `audio/gameplay.wav` ← صوت أثناء اللعب (عند جمع عنصر).
3. `audio/end.wav` ← صوت نهاية الجولة (فوز/خسارة/انتهاء).

> إذا لم تكن هذه الملفات موجودة، اللعبة ستعمل بدون أصوات تلقائيًا.
