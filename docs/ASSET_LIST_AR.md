# الملفات الصوتية (اختيارية) — أضفها يدويًا

> مهم جدًا: الملف يجب أن يكون WAV حقيقي بصيغة RIFF وليس MP3 مع تغيير الامتداد.

ضع الملفات داخل المجلد: `audio/`

1. `audio/start.wav`  ← صوت بدء اللعبة.
2. `audio/gameplay.wav` ← صوت جمع العناصر داخل اللعب.
3. `audio/end.wav` ← صوت نهاية الجولة (فوز/خسارة).

إذا لم تضف هذه الملفات، اللعبة تعمل عادي بدون صوت.
